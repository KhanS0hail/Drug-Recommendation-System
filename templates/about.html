About the Drug Recommendation System

Overview of the System

The Drug Recommendation System aims to enhance the prescription process by providing accurate predictions of diseases based on user-input symptoms. By leveraging advanced machine learning algorithms, the system assists healthcare professionals in making informed decisions, ultimately improving patient outcomes.

How it Works

Users input their symptoms into the system, which utilizes the Support Vector Machine (SVM) algorithm to predict potential diseases. This algorithm is trained on a CSV dataset obtained from Kaggle, along with other datasets that provide information on medication, workout recommendations, diet plans, and precautionary advice. The system then displays relevant details about the predicted disease and offers tailored recommendations.

Benefits

  • For Doctors: Enhances the accuracy of disease predictions, allowing for more effective treatment plans.
  • For Patients: Empowers patients with personalized recommendations for medication and lifestyle adjustments, promoting better health management.
  • For Pharmacists: Assists in ensuring the correct medications are dispensed based on the system's recommendations, streamlining the dispensing process.

Technology Stack

The system is built using Python and Flask as the framework. It utilizes scikit-learn for machine learning models, along with other essential libraries such as Pandas for data manipulation. The frontend is developed using HTML, CSS,Bootstrap, and JavaScript.
